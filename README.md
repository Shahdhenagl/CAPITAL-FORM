# عاصمة الكون للمصاعد — فورم + لوحة تحكم

موقع بسيط فيه:

- **فورم عامة** (الصفحة الرئيسية `/`) للعميل: الاسم، الجوال، الواتساب، نوع المنشأة، العنوان، وتحديد الموقع على الخريطة (OpenStreetMap مجاني + توليد لينك خرائط جوجل تلقائياً).
- **لوحة تحكم** (`/dashboard`) محمية بكلمة سر: بتعرض كل الطلبات لحظياً، وتقدري منها تحددي موعد الزيارة وتبعتيه للعميل عبر **واتساب** بضغطة واحدة.
- **إشعار تليجرام** فوري بكل طلب جديد على البوت بتاعك.
- قاعدة البيانات: **Supabase**.

---

## 1) تشغيل المشروع محلياً

```bash
npm install
cp .env.example .env.local   # ثم املئي القيم
npm run dev
```

افتحي: <http://localhost:3000> للفورم، و <http://localhost:3000/dashboard> للوحة التحكم.

---

## 2) إعداد قاعدة البيانات (Supabase)

1. اعملي مشروع جديد على <https://supabase.com>.
2. من القائمة: **SQL Editor → New query**، الصقي محتوى ملف `supabase.sql` واضغطي **Run**.
3. هاتي بيانات الربط من **Project Settings**:
   - `SUPABASE_URL` ← من **Data API → Project URL**
   - `SUPABASE_SERVICE_ROLE_KEY` ← من **API Keys → service_role** (سري جداً، الخادم بس بيستخدمه).

---

## 3) إعداد بوت تليجرام

1. كلّمي **@BotFather** على تليجرام واعملي بوت جديد بأمر `/newbot` وخدي **التوكن** ← `TELEGRAM_BOT_TOKEN`.
2. ابعتي رسالة لأي حد/جروب، وعشان تجيبي **chat id**:
   - أسهل طريقة: كلّمي **@userinfobot** هيديكي رقم حسابك ← `TELEGRAM_CHAT_ID`.
   - أو لو عايزة الإشعار يجي على جروب: ضيفي البوت للجروب، وافتحي
     `https://api.telegram.org/bot<التوكن>/getUpdates` بعد ما تبعتي رسالة في الجروب، وخدي `chat.id` (بيكون رقم بالسالب للجروبات).

> الإشعار بيتبعت تلقائياً مع كل طلب جديد، وفيه كل بيانات العميل + لينك الموقع.

---

## 4) متغيرات البيئة (Environment Variables)

كل المتغيرات موجودة في `.env.example`. محتاجة تحطيها في `.env.local` محلياً، وفي إعدادات Vercel وقت الديبلوي:

| المتغير | الوصف |
|---|---|
| `SUPABASE_URL` | رابط مشروع Supabase |
| `SUPABASE_SERVICE_ROLE_KEY` | مفتاح service_role (سري) |
| `DASHBOARD_PASSWORD` | كلمة سر الدخول للداش بورد |
| `SESSION_SECRET` | أي نص عشوائي طويل لتأمين الجلسة |
| `TELEGRAM_BOT_TOKEN` | توكن بوت التليجرام |
| `TELEGRAM_CHAT_ID` | معرّف الشات اللي هتوصله الإشعارات |

---

## 5) النشر على Vercel

1. ارفعي المشروع على GitHub، أو استخدمي `vercel` CLI.
2. على <https://vercel.com> اعملي **New Project** واختاري الريبو.
3. في **Settings → Environment Variables** حطي نفس المتغيرات اللي فوق.
4. **Deploy**. اللينك اللي هيطلع هو لينك الفورم اللي تبعتيه للعملاء، والداش على `/dashboard`.

## ملاحظات

- الخريطة مجانية (OpenStreetMap) من غير أي مفتاح. العميل بيحدد بالضغط على الخريطة أو بزر «موقعي الحالي»، والنظام بيحفظ الإحداثيات + لينك خرائط جوجل.
- لينك الواتساب بيتعامل مع الأرقام المصرية (اللي بتبدأ بـ 0) ويحوّلها تلقائياً لصيغة دولية (20).
- لتغيير الموقع الافتراضي للخريطة، عدّلي `DEFAULT_CENTER` في `components/LocationPicker.jsx`.
